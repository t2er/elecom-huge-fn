# Elecom HUGE : setting fn buttons

## install:
- xbindkeys
- xautomation (xte)
- xdotool

## add to ~/.xbindkeysrc

## reload the config file
```
killall xbindkeys
xbindkeys
```

## add to autostart 
```
xbindkeys
```

## key number
1 - left button
2 - middle button
3 - right button
6 - middle button down
7 - middle button up
8 - back
9 - forward
10 - Fn1
11 - Fn2
12 - Fn3
